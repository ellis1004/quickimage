⏱️ 什麼是番茄工作法？

番茄工作法（Pomodoro Technique）由 Francesco Cirillo 於1980年代後期創立。它提倡：專注工作 25 分鐘，隨後休息 5 分鐘。這是一個循環。每完成 4 個循環，進行一次 15~30 分鐘的長休息。這種節奏有助於緩解大腦疲勞並維持高效專注狀態。

🔊 為什麼整合 Web Audio 音效？

科學研究指出，穩定的大自然聲音（如雨聲、風聲）可以有效阻隔外界的突發雜音，並降低大腦對周遭環境的敏感度，進而幫助進入「心流（Flow）」狀態。本站採用 Web Audio API 進行本地音頻實時合成，不消耗您的網路流量，亦沒有音樂版權問題。
